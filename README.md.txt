⚡ Tesla Coil Using 2N2222 Transistor

 📌 Project Overview

This project demonstrates a **low-power Tesla Coil circuit** built using a **2N2222 NPN transistor**, designed to illustrate the principles of:

- Electromagnetic induction  
- Resonant transformer action  
- Wireless energy transfer  
- High-frequency oscillation  

The system operates using a simple **9V DC supply**, making it suitable for **educational and experimental electronics learning**.

 🎯 Objective

- Build a compact Tesla coil using minimal components  
- Demonstrate wireless power transfer effects  
- Understand high-frequency transformer behavior  
- Study transistor-based self-oscillating circuits  

 ⚙️ System Principle

The circuit works based on a **self-oscillating feedback loop**:

1. A 2N2222 transistor switches current through the primary coil  
2. The primary coil generates a magnetic field  
3. The secondary coil (high turns) induces high voltage via electromagnetic coupling  
4. Continuous feedback sustains oscillation  
5. High-frequency AC output is produced at the secondary coil  


 🔩 Hardware Components

| Component | Specification |
|----------|--------------|
| Transistor | 2N2222 (NPN) |
| Power Supply | 9V DC Battery |
| Resistor | 2.2kΩ / 22kΩ (depending design) |
| Primary Coil | 3–7 turns thick copper wire |
| Secondary Coil | ~300 turns thin enamel copper wire |
| Switch | SPST ON/OFF |
| Base Structure | PVC pipe (coil former) |
| LED (optional) | Indicator |

 🧲 Coil Construction

 Secondary Coil
- Wind ~300 turns of 28 AWG enamel copper wire
- Use a 9 cm PVC pipe as the coil former
- Ensure tight, uniform winding without overlap
- Secure both ends properly

Primary Coil
- 3–7 turns of thick copper wire
- Placed near bottom of secondary coil
- Should not touch secondary winding

---

 🔌 Circuit Operation

 Step 1: Power Input
- 9V battery supplies DC voltage
- Switch controls ON/OFF operation

Step 2: Oscillation
- Base resistor triggers transistor switching
- Transistor operates in switching mode

 Step 3: Magnetic Coupling
- Primary coil generates alternating magnetic field
- Secondary coil picks up induced voltage

Step 4: Output
- High-frequency AC appears at secondary coil
- Can illuminate fluorescent lamps wirelessly

 ⚠️ Safety Precautions

- Do NOT touch coil while operating  
- Keep away from sensitive electronics  
- Avoid prolonged operation (transistor heating risk)  
- Operate in dry, ventilated environment  
- Use insulated wires properly

 🔧 Troubleshooting

❌ No Output
- Check transistor pin orientation  
- Verify battery voltage  
- Inspect coil winding direction  

❌ Transistor Heating
- Add heatsink  
- Reduce operating time  
- Check base resistor value  

❌ Weak Output
- Improve coil winding quality  
- Adjust primary-secondary distance  
- Ensure tight coupling
